# SocNetworkDemo
## Чтобы запустить, в командной строке нужно:
### Клонировать репозиторий
```
    git clone https://github.com/Potagashev/SocNetworkDemo.git
```
### Установить зависимости
```
  pip install -r requirements.txt
```
### Запустить сервер
```
python manage.py runserver
```
### Перейти по ссылке
```
http://127.0.0.1:8000/
```
